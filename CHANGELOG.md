# Changelog

All notable changes to Cognitia will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [Unreleased]

### Added
- REST API with FastAPI
- PostgreSQL persistence layer
- WebSocket real-time updates
- Multi-organization support
- Visualization dashboard

## [1.0.0] - 2025-06-24

### Added
- Autonomous corporate evolution engine
- 5 departments: Executive, Finance, R&D, Marketing, HR
- Agent system with morale, skill, memory, attrition
- Internal economy: C-credits, budget allocation, ROI tracking
- External market simulation: demand, competition, shocks
- Analytics: efficiency, innovation, conflict, adaptability metrics
- Deterministic replay via seed
- ~23k ticks/second performance
- Professional documentation

### Architecture
- Modular department system
- Per-tick decision loops
- Inter-department conflict generation
- Dynamic hiring/firing via HR
